3. Creating an Editorial Plan ‚Äì Without a Goal, Content Goes Nowhere

Before planning, define your primary goal. Without a goal, the AI produces random output.

Possible goals:

  • Visibility & reach
  • Lead generation / inquiries
  • Building expertise
  • Building trust

Structure of the Editorial Plan Prompt

This prompt consists of clear building blocks that you can adapt to your own needs at any time:

  1. Goal definition
    Now create a one-month editorial plan with the goal of {{GOAL}}.
  2. Frequency & scope
    I want to publish {{NUMBER}} posts per week / per month.
  3. Timing (optional, but recommended)
    My best days and times to publish are {{DAY + TIME}}, e.g. Tuesday morning and Thursday afternoon.
  4. Output format
    Please present the editorial plan as a table and make it available for download so I can use it as an Excel file.

üëâ Important: You can explicitly request additional columns in the prompt, for example:

  • publication date
  • channel / platform (e.g. LinkedIn, Instagram, Facebook)
  • status (planned, published, revised)
  • call-to-action

If you do not specify these in the prompt, you can easily add them later directly in Excel.

4. Style Is Not an Accident ‚Äì It Is a Decision

Before finalizing content, define your style consciously ‚Äì not based on gut feeling, but systematically.

Tool: ChatGPT (or another AI of your choice)

üîó Go to ChatGPT: https://chatgpt.com

üëâ Note: Again, use the AI you are most familiar with.

Full Prompt for Style Analysis (do not shorten)

Use this prompt in full and without any changes or omissions. Every sentence is relevant.

**Prompt:**

You are an AI coach for writing style and tone of voice.
Your task is to analyze the style of the following text so precisely that it results in a reusable style prompt that can be used to consistently create new texts in the same style.

Procedure:

1. Style Analysis
Analyze the text according to the following characteristics:
communicative stance (e.g., explanatory, narrative, motivating, opinionated)
target audience and language level (e.g., technical, approachable, mixed)
tone of voice (e.g., direct, emotional, factual, inspiring)
sentence style (e.g., short impulses vs. complex sentence structures)
linguistic rhythm (e.g., pauses, emphasis, rhetorical questions)
word choice (e.g., active vs. passive, strong verbs vs. nouns)
structure (paragraphs, lists, examples, dramaturgy)
personal language signatures (e.g., recurring terms, anglicisms, typical phrases, signature words like ‚ÄúYes,‚Äù ‚ÄúNope,‚Äù ‚ÄúExactly,‚Äù intensifiers such as ‚Äúquite honestly,‚Äù emojis)
linguistic taboos or deliberately avoided words

If personal language signatures are not clearly identifiable in the text, ask up to three targeted follow-up questions:
Do you use specific favorite words or recurring phrases?
Do you often use anglicisms or brand-specific terms?
Are there words or expressions you deliberately avoid?

2. Style Fingerprint
Summarize the style in 8‚Äì10 well-detailed points. Structure:
‚Äì perspective and form of address
‚Äì tone of voice
‚Äì sentence style
‚Äì linguistic rhythm
‚Äì word choice and patterns
‚Äì structure and composition
‚Äì personal language signatures
‚Äì do/don‚Äôt rules for the style

3. Create the Style Prompt
Based on the style fingerprint, develop a reusable prompt (200‚Äì300 words) that can be used to generate texts in the analyzed style.
The prompt should:
be written in the language of the original text
include placeholders (e.g., {{Topic}}, {{Goal}}, {{Format}}, {{TargetAudience}}, {{Tone}})
define clear style rules
take personal language signatures into account
include a short quality check (checklist)

4. Output Format
Provide exactly these two parts:
A) Style Fingerprint (list format, concise)
B) Style Prompt

Here is the text to be analyzed:
<<<
{{TEXT}}
> > >

Where does the text come from?

Use your own already successful content, for example:

  • high-performing LinkedIn posts
  • newsletter texts
  • blog articles

This can also be a successful post from any other platform. What matters is that the style already works and can be reproduced.
